# Home Lab - Personal Network Security & Infrastructure

A self-hosted home lab project documenting the design and implementation of a localized, secure, network supporting multiple devices, controlled communication, and layered security. The goal is to build practical understanding of networks, infrastructure, and security architecture through hands-on construction.

## Project Goals

- Establish secure, controlled communication between personal devices.
- Implement layered security across the network stack (VPN, firewall, DNS, monitoring).
- Build a virtualized environment for ongoing security practice.
- Integrate self-hosted device control (smart home) under the same security model.
- Document each pase as a portfolio of applied training.

## Approach

Each phase is built deliberately, documented honesty including limitations, and used to inform the design of the next. This progression-based approach prioritizes depth of udnerstanding over breadth of tools.
